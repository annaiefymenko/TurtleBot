# WS25-Turtle Bot Take Control 
# ROS - based Turtle Bot project

## Übersicht 

git clone https://github.com/annaiefymenko/TurtleBot.git

Dieses Projekt implementiert eine **multimodale Steuerung für einen TurtleBot**:
- Steuerung über **Python ROS2 Nodes**
- **Spracheingabe** mit Whisper (Speech-to-Text)
- **Sprachausgabe** (Text-to-Speech)
- **Web-Interface** mit React zur Fernsteuerung

Das Ziel ist, dass der Roboter **interaktiv auf Sprachbefehle reagiert**, gleichzeitig sicher navigiert und über die Webseite gesteuert werden kann.

---

## Features

- ROS2 Nodes zur Robotbewegung und Sicherheit
- Sprachsteuerung über Whisper
- Sprachausgabe über TTS Node
- React-Webseite zur Steuerung
- Echtzeit-Kommunikation zwischen Frontend und ROS Nodes via WebSockets
